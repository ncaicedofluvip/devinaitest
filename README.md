# DevinAI Test - Angular Application

An Angular application bootstrapped with Angular CLI v21.

## Requirements

- Node.js v20 or higher (tested with v22.12.0)
- npm v10 or higher

## Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/ncaicedofluvip/devinaitest.git
cd devinaitest
npm install
```

### Development Server

Run the development server:

```bash
npm run start
```

Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

### Build

Build the project for production:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

### Running Tests

Run unit tests:

```bash
npm run test
```

## Project Structure

The application includes the following routes:

- `/` - Home page
- `/report` - Report page (placeholder)

## Main Commands

| Command | Description |
|---------|-------------|
| `npm install` | Install dependencies |
| `npm run start` | Start development server |
| `npm run build` | Build for production |
| `npm run test` | Run unit tests |
| `npm run watch` | Build and watch for changes |
